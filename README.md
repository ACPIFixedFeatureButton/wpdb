# wpdb
wpdb - Windows Phone Database

Basically still LumiaDB but for other OEMs that also happen to make Windows Phone (not just Nokia and Microsoft).

**Contributors note:**
If you have double quotes in the guides or literally any JSON field, please escape them with backlashes. You can view how to do it [here](https://www.geeksforgeeks.org/javascript/how-to-escape-double-quotes-in-json/).
